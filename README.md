## 1. 알고리즘이란? (07.01)

#### 알고리즘의 정의
어떠한 문제를 해결하기 위해 정해 놓은 일련의 절차

#### 파이썬 스타일 가이드 PEP 8
클래스명은 CamelCase, 함수명은 snake_case로 쓸 것을 권장

#### 파이썬의 변수
- 변수는 객체를 참조하는 객체에 연결된 이름에 불과합니다.
- 모든 객체는 메모리를 차지하고, 자료형뿐만 아니라 식별 번호(identity)를 가집니다.
  ```
  n = 17
  id(17) # 140710805205776
  id(n) # 140710805205776
  ```
